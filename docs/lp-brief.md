# GRAPE — LP Design Brief

## 1. プロダクト概要

**GRAPE** は、日本の「複数推し活」ファン向けの **ライブ参戦記録アプリ**。

- チケット抽選 / 入金期限 / 当日支払い / 発券 / 完了 まで **7段階で追跡**
- **ダブルブッキング検知** · **週次合算** · **アーティスト別統計** など、複数推しに最適化
- **URL 1枚貼れば AI がイベント情報を自動入力**（Premium）
- 買い切り ¥480（リリース記念 / 通常 ¥980）· サブスクなし
- 本 LP は公開前ローンチ用の最初の LP

## 2. ターゲット

### 対象ユーザーの幅（性別・年齢を問わず）
「ライブに行く」習慣があり、**複数の推しの参戦履歴を記録したい人すべて**。
- 10代の地下アイドルファンから、50〜60代のバンド好きの夫婦まで
- ジャンル: アイドル / バンド / VTuber / 声優 / 俳優 / 舞台 / アニソン etc.
- 男性・女性・ノンバイナリー問わず
- 動機はデモグラじゃなく**行動特性**（月に複数回ライブに行く／複数推しを同時に追う／チケット期限を管理したい／思い出を記録したい）

### 代表的なペルソナ（※ユーザー層はこれに限らない）
「こはるさん」26歳 女性 都内在住 会社員
- 推し: 地下アイドル 1〜2組 + インディーバンド 2〜3組 + VTuber 1人
- 月の推し活予算: ¥20,000〜40,000、参戦頻度: 月 4〜8 回
- 悩み:
  - チケット抽選の発表日・入金期限を複数同時に覚えていられない
  - 同じ日に複数のライブが被ったときに気付けない
  - 「今月いくら使った？」「このアーティスト何回行った？」をメモ帳じゃ把握できない
  - 推し活の軌跡を美しく振り返りたい（Spotify Wrapped 的）

**差別化**: 「複数推しネイティブ」「UIの情緒感」「URL1枚で自動入力」

## 3. ブランドパーソナリティ

### Tone
**Calm, personal, and slightly premium.** The aesthetic is earthy and organic — not flashy. Think "a beautiful notebook you carry to every concert." 温かみ・情緒的・洗練・信頼感・日本的「推し活」文化。

### NG
- 派手な紫グラデーション（紫系だが上品に）
- 「効率化」「タスク管理」的な硬い訴求
- インスタ風のキラキラ表現、男性向けガジェット風、女性向けフェミニン寄り
- ダークモード基調（off-white が主役）
- 特定の性別・年齢に寄ったビジュアル（例: ピンク系パステル、スーツ姿のビジネスマン、等）

## 6. デザインシステム

### Color Palette（GRAPE プリセット / 紫系、スクショと統一）
- Background: `#F5F3FB`
- Section BG: `#EDE9F5`
- Primary Text/Icon: `#3D1A78` (deep purple)
- Primary Muted: `#9878CC`
- Accent (CTA/warnings): `#C04890` (warm pink-magenta)
- Border: `#C0B2D8`
- White: `#FFFFFF`
- Premium BG: `#1C0F42` (dark purple, Premium カードのみ)
- Premium Gold: `#F5C850` (Premium カードのみ)

### Typography
- Latin/Numbers: **Google Sans** (local)
- Japanese: **Noto Sans JP** + Google Sans
- Weight は **700 (Bold) と 400 (Regular) のみ**。500 / 600 / 300 は使わない。

| Role | Size | Weight |
|------|------|--------|
| Display (hero) | 56–72px | 700 |
| H2 section | 36–48px | 700 |
| H3 card | 20–24px | 700 |
| Body | 15–17px | 400 |
| Caption | 12–13px | 400 |

### Shape
- Cards: 8–14px
- Pills / Buttons: 999px
- Phone frame mockup: 44px (iPhone-like)

### Layout
- Max content width: 1200px
- Section vertical padding: 80–120px
- Mobile responsive 必須（ブレイクポイント 768 / 1024）

### Animation
- スクロール連動フェードイン（subtle、200–360ms）
- CTA ホバーで ripple
- 過度なアニメーションは避ける

## 11. ブランドアセット（添付済み・必ず使用）

**GRAPE ロゴ（wordmark）と App アイコンをこのプロンプトに添付済みです。LP 内で必ず活用してください。**

### GRAPE ロゴ（wordmark）
- **ホワイト版**（白い "Grape" 英字） → ダーク背景（深紫・画像オーバーレイ）で使用
- **紫版**（`#382075` の "Grape" 英字） → ライト背景（白・off-white）で使用
- **縦横比 584:190 を必ず保持**（歪めない・切り抜かない）
- 配置目安:
  - ヘッダー左上: 高さ 24〜32px
  - フッター: 高さ 36〜44px
  - ヒーローで大きく使う場合はアプリアイコンと組み合わせて控えめに

### App アイコン（正方形・iOS スタイル）
- 深紫 `#382075` 背景 + 白い四光星（sparkle）のアイコン
- **角丸 22%（`border-radius: 22%`）で iOS squircle 風に表示**
- 配置目安:
  - ヒーロー上部に GRAPE ロゴと **セットで** 配置（アイコン 72〜96px）
  - 「ダウンロード」CTA セクションで App Store / Google Play バッジと並べる
  - フッターのアプリ情報ブロック
- ソフトな外側グローを足すと Premium 感が出る:
  `box-shadow: 0 12px 48px rgba(56, 32, 117, 0.45)`

### ロゴ紫と LP 紫の使い分け
- ロゴ色 `#382075` と LP の Primary `#3D1A78` は **近似のダーク紫**
- **両色を同じグラデーションレイヤーで共存**させて深み・奥行きを演出してよい
  例: `linear-gradient(135deg, #3D1A78 0%, #382075 50%, #2A1658 100%)`
- ロゴを置く背景色は **ロゴ色 `#382075` 自身を避け**、わずかに濃い／薄いトーンで対比させる

### 要点チェック
- [ ] ヘッダー・ヒーロー・フッターの **少なくとも 2 箇所以上** にロゴ / アイコンを登場させる
- [ ] ダーク背景ではホワイト版、ライト背景では紫版を必ず使い分ける
- [ ] アイコンは角丸 22%（iOS squircle）で統一
- [ ] ロゴを変形・色変更・切り抜きしない（SVG 原形維持）
